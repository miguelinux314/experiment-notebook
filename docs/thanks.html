Thanks

Big thanks to all these people/projects without which enb would not be possible:

Core libraries

The enb core is firmly supported by

  • pandas

  • matplotlib

  • numpy

  • pathos

and, for multi-computer processing, by

  • ray

  • ssh

  • sshfs

  • vde2

Several other are also essential and their contribution greatly appreciated. Please see the requirements in setup.py for more details.

Special thanks to all contributors to python itself. Without them, this library would not be here.

Plugin libraries

Many external libraries are employed by enb’s plugin system. Each offered plugin contains full reference to their respective authors, who also have enb’s eternal gratitude.

In addition, credits to (at least) the following people are due:

  • Ayushman Dash and Zhou Wang for their SSIM code

  • Yoel, for this great post.

  • Geeks for Geeks for their snippet on CamelCase handling

  • Lorenzo for this post on argument parsing.

  • cgohlke for this post on easy PGM parsing.
