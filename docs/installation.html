Installation¶

To start using the enb library,

  1. Download the last version of the code.
  2. Unzip it
  3. Run the setup.py script

Under linux, this can be accomplished with the following lines:

wget https://github.com/miguelinux314/experiment-notebook/archive/master.zip
unzip master.zip
cd experiment-notebook-master
python3 ./setup.py install --user # or sudo python3 ./setup.py install
